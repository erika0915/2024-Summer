# 대학생 전용 금융 상품 비교 서비스,EWHA INVESTIGATION

## 📌 프로젝트 소개
대학생들을 위한 [예금/적금/대출/체크카드] 5개의 은행 상품들을 한 눈에 볼 수 있도록 정리하여 보다 쉽게 대학생들이 금융적 혜택을 누릴 수 있도록 도와준다.
